# The-3rd-4th-Republic-MBC

本仓库为前日海字幕组组员用于存放MBC韩剧《第三共和国/第四共和国》的临时字幕剧集地址。
由于现日海字幕组利用韩剧字幕进行大规模恶意吸取群员众筹资金的行为，其对本剧影响十分恶劣，为肃正影响并划清界限，特上传此仓库。
